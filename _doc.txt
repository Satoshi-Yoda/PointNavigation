Программа реализует симуляцию перемещения газонокосилки с авто-определением местоположения по ключевым точкам (специальным столбикам, установленным на земле).
Технически определение местоположения осуществляется с использованием снимков, сделанных установленной на газонокосилке камерой.
Старт симуляции осуществляется нажатием кнопки Snake, которая запускает газонокосилку ездить "змейкой", скашивая круглую область в центре.

Нажатие кнопки Error map приводит к запуску просчёта достаточной точности для разных областей.
Зелёное - хорошая точность. Красное заштрихованное - плохая точность.
При наведении на красную/зелёную область рядом с указателем мыши выводится ресчётное значение средней погрешности для этой точки.

Точность определения местоположения зависит от двух факторов: от схемы расположения ключевых точек и от разрешения камеры, используемой для навигации.
Смоделированы три разные по разрешающей способности камеры с разрешением по горизонтали от 320 пикселей до 1920 пикселей.
Ключевые точки обозначены числами. Выбирать их расположение можно либо одним из предложенных способов, либо вручную.
